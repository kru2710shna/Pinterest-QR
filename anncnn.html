🧠 ANN & CNN Project

Artificial Neural Networks & Convolutional Neural Networks

Overview

This project explores two powerful deep learning techniques: Artificial Neural Networks (ANN) for churn prediction and Convolutional Neural Networks (CNN) for image classification. Both models achieved 87% accuracy in their respective tasks.

Artificial Neural Networks (ANN)

  • 📦 Import dependencies & load Kaggle dataset.
  • ⚙️ Feature engineering using standard scaling.
  • 📊 Split dataset into training & testing sets.
  • 🧩 Construct and initialize ANN structure.
  • ➕ Add layers for optimal performance.
  • 📈 Train model and evaluate with accuracy_score.
  • ✅ Achieved accuracy: 87%.

Convolutional Neural Networks (CNN)

  • 📦 Import dependencies & prepare CIFAR-10 dataset.
  • 🔍 Verify and preprocess data.
  • 🏗️ Build convolutional base and add dense layers.
  • ⚙️ Compile and train model.
  • 📊 Evaluate model performance.
  • ✅ Achieved accuracy: 87%.

🩻 Chest X-ray Classification

Multi-label classification using frontal chest X-ray images

Overview

This project focuses on classifying multiple pathological conditions using chest X-ray images. Leveraging image data and multi-label classification techniques, it aims to assist clinicians in recognizing disease markers from scans.

Workflow Steps

  • 🖼️ Data Acquisition & Preprocessing (resize, normalize, augment).
  • 🔍 Feature Extraction / CNN layers.
  • 🧩 Building the multi-label classifier.
  • 🧪 Training, validation & hyperparameter tuning.
  • 📈 Evaluate using AUC, accuracy, recall, precision, etc.

Technologies Used

Python | TensorFlow / Keras | OpenCV | Deep Learning

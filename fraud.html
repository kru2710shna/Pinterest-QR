💳 Fraud Detection with BigQuery ML

Google Cloud | BigQuery ML | Financial Data

Overview

This project leverages Google BigQuery ML to analyze financial transactions and detect fraudulent activity. It utilizes large-scale public financial datasets and builds predictive models to flag high-risk transactions.

Objectives

  • 📊 Data Exploration: Load and inspect BigQuery financial datasets.
  • 🔍 Feature Engineering: Extract transaction type, balances, and time features.
  • 🤖 Model Training: Train fraud detection models using BigQuery ML.
  • 📈 Evaluation: Validate predictions with fraud/non-fraud labels.

Tech Stack

BigQuery ML | Google Cloud Platform | SQL | Jupyter Notebook

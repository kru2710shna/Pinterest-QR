📱 HobbyTrade

Marketplace for trading hobbies, ideas, and skills

🔍 Overview

HobbyTrade is a full-stack mobile platform enabling hobbyists to explore, trade, and collaborate. With React Native and a MongoDB backend, it supports multi-step onboarding, guest browsing, Stripe payments, and deep-link enabled profile navigation.

🛠️ Tech Stack

  • React Native + MongoDB
  • Node.js + Express.js + JWT Auth
  • Stripe API + Deep Linking

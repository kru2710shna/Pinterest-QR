Overview

HobbyTrade is a full-stack mobile platform enabling hobbyists to explore, trade, and collaborate. With React Native and MongoDB backend, it supports multi-step onboarding, Stripe payments, and deep-link enabled profile navigation.

Tech Stack

  • Node.js, Express, JWT
  • Stripe API, Deep Linking

HobbyTrade Platforms

Marketplace for trading hobbies + ideas

🔍 Overview

HobbyTrade is a full-stack marketplace that allows users to buy, sell, and showcase their hobbies. Built with React Native and MongoDB, it features multi-step onboarding, guest mode browsing, Stripe integration, and collaboration tools.

🛠️ Stack

  • React Native
  • MongoDB + Node.js
  • Stripe API
  • JWT Auth + Deep Linking

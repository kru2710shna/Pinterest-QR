🩺 Kidney Disease Classification

Predict kidney disease using ML + Deep Learning pipeline

Overview

This project classifies kidney disease from medical data, using machine learning and deep learning models. It provides predictions and interpretable insights to assist in early detection.

Workflow Steps

  • 📊 Data preprocessing — missing data, normalization, encoding.
  • 🔍 Feature engineering and selection.
  • 🧠 Model training — classical ML + neural networks.
  • 📈 Model evaluation using metrics (accuracy, ROC, recall, etc.).

Technologies Used

Python | Pandas | Scikit-learn | TensorFlow / Keras

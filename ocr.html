📄 Document Text Detection (OCR)

Extracting text from images & documents using ML, OpenCV, Tesseract

Overview

This project implements OCR (Optical Character Recognition) to detect and extract text from images and documents. It uses OpenCV for image preprocessing and Tesseract for text recognition, with a Flask backend to power the web interface.

Workflows & Steps

  • 🖼️ Data Preparation: Load and preprocess images (resize, denoise).
  • 🔍 Text Region Detection: Use contours, thresholding, bounding boxes.
  • 🧰 Text Extraction: Use Tesseract OCR to extract characters/words.
  • 🛠️ Postprocessing: Clean extracted text (remove noise, correct spacing).
  • 🌐 Web Backend: Flask handles image upload, triggers OCR pipeline, returns results.

Technologies Used

Python | OpenCV | Tesseract | Flask | Gunicorn

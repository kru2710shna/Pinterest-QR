🏅 Olympic Medal Predictor

Machine Learning | Jupyter Notebook | Data Analysis

Overview

This project predicts the exact number of medals a country will win in the Olympics using Machine Learning. The goal was to minimize prediction error with low Mean Absolute Error (MAE).

Steps

  • 📊 Data Collection: Used Olympic datasets with country-level features.
  • 🔎 Feature Engineering: Cleaned and structured datasets for ML modeling.
  • 🤖 Model Training: Applied regression techniques for medal prediction.
  • 📈 Evaluation: Measured accuracy using MAE to validate results.

Tech Stack

Python | Pandas | Scikit-learn | Jupyter Notebook

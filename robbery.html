Robbery Detection AI

YOLOv8-based system for detecting loitering, aggression, and theft in real-time

🔍 Overview

This AI system analyzes real-time surveillance feeds to detect suspicious human behaviors like loitering, aggressive movement, and object theft using YOLOv8, OpenCV, and pose estimation. Includes color-coded escalation logic and visual overlays.

🛠️ Stack

  • YOLOv8 + Ultralytics
  • OpenCV + Python
  • Pose Estimation + Keypoint Analysis
  • Modular architecture with real-time alerts

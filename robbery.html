Robbery Detection AI

Real-time behavior monitoring using YOLOv8 & pose estimation.

Overview

This system uses YOLOv8 and OpenCV to analyze surveillance footage in real-time for detecting loitering, aggressive gestures, and object theft. Features pose keypoint logic, alert escalation (green → yellow → red), and modular detection layers.

Tech Stack

  • YOLOv8 + Ultralytics
  • OpenCV + Python
  • Pose Keypoints + Aggression Detection
  • Real-time alert pipeline

Overview

SmokeCrypt is a novel cryptographic entropy generator that utilizes incense smoke patterns, optical flow, and OpenCV for producing secure OTPs. Built entirely in Python with real-time video entropy sampling.

Tech Stack

  • Python + OpenCV
  • HMAC, Optical Flow
  • Entropy windowing

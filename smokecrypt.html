SmokeCrypt 🔒

Entropy from smoke → real cryptography.

🔍 Overview

SmokeCrypt is a novel entropy generation method using incense smoke + OpenCV + HMAC to create OTPs. Built in Python with real-time video processing.

🛠️ Stack

  • Python, OpenCV, HMAC
  • Optical flow, entropy pool windowing
